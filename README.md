# Malek ID

Malek Imam ID Card

## Flutter Application

Project contain one main <b>widget</b> which is the Card ID and this widget contains the following widgets:<br>

> - App Bar Widget
> - Avatar Widget
> - Name (Text) Widget
> - Job title (Text) Widget
> - Icon + Text (Row) Widget
